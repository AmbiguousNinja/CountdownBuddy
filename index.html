Countdown Buddy

Companion app for the popular British gameshow Countdown


Countdown Rules

  1. Using the selected numbers, the target must be reached
  2. Players choose 6 numbers between 1 and 10 and from the set, {25, 50, 75, 100}.
  3. A target number between 101 and 999 is then randomly generated.
  4. Using only addition, subtraction, multiplication and division, the player must get as close as possible to the target number.
  5. Each number from the selected set may only be used once
  6. Decimal and negative numbers are not allowed during any calculation.
